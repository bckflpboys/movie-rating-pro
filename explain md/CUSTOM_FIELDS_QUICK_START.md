When you view a saved rating:
- Click on any rating card
- Scroll to **"Additional Information"**
- See all your custom field values!

## üîÑ Managing Fields

### Edit a Field
1. Go to Settings
2. Change the label or type
3. Save Settings
4. Your field is updated!

### Delete a Field
1. Go to Settings
2. Click "Delete" next to the field
3. Save Settings
4. Field removed from form!

**Note:** Deleting a field doesn't delete old data - values remain in previously saved ratings.

## ‚ú® Tips & Tricks

1. **Keep it Simple**: Start with 2-3 fields, add more as needed
2. **Use Dropdowns**: Great for consistent data (genres, platforms, etc.)
3. **Long Text for Reviews**: Use "Long Text" for detailed notes
4. **Optional Fields**: All custom fields are optional - skip them if you want
5. **Persistent Data**: Your fields and values are saved forever!

## ‚ùì FAQ

**Q: How many custom fields can I add?**
A: Unlimited! But we recommend keeping it under 10 for usability.

**Q: Will my custom fields sync across devices?**
A: Not currently - they're stored locally on each browser.

**Q: Can I reorder my custom fields?**
A: Not yet - they appear in the order created.

**Q: What happens if I delete a field?**
A: It's removed from the form, but old ratings keep their values.

**Q: Can I make a field required?**
A: Not yet - all custom fields are optional.
